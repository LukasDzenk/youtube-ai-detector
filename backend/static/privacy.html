Privacy Policy

Last updated: March 6, 2025

The short version

YouTube AI Detector does not collect personal data. There are no accounts, no emails, no cookies, and no tracking. We store the bare minimum needed to count reports.

What data we store

  • Anonymous install ID — when you install the extension, a random UUID is generated locally on your device. This ID is sent with each report so the server can tell "one person reported twice" from "two people reported once." It cannot be linked back to you.
  • Video reports — when you flag a video, we store the YouTube video ID, the anonymous install ID, and a timestamp. That's it.

We do not store your name, email, IP address, browsing history, watch history, or any other personal information.

Local storage

The extension uses chrome.storage.local to save your preferences (color theme, tooltip setting) and your report history on your device. This data never leaves your browser.

Network requests

The extension communicates only with our backend API to:

  • Check if a video has been reported (when you visit a YouTube watch page)
  • Submit or undo a report (when you click the flag button)

No data is sent to any third party. The extension does not run on any site other than youtube.com.

Website analytics

This landing page uses Plausible Analytics, a privacy-friendly, cookie-free analytics tool. It collects no personal data and is fully GDPR compliant. The extension itself has no analytics whatsoever.

Open source

The entire codebase — extension, backend, and database schema — is open source on GitHub. You can audit exactly what data is collected and how it is handled.

Data retention

Report data is stored indefinitely to maintain community flags. Since no personal information is attached, there is nothing personally identifiable to delete. If you uninstall the extension, your local data is automatically removed by Chrome.

Children's privacy

YouTube AI Detector is not directed at children under 13. We do not knowingly collect data from children.

Changes to this policy

If this policy changes, the updated version will be posted here with a new date. Since we don't have your email, we can't notify you directly — check back if you're curious.
